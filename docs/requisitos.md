# Requisitos do Sistema – Conecta Aluno

## Requisitos Funcionais

RF01 – O sistema deve permitir que usuários se cadastrem.

RF02 – O sistema deve permitir que usuários façam login.

RF03 – O sistema deve permitir que o usuário crie um perfil com suas habilidades.

RF04 – O sistema deve permitir que usuários cadastrem serviços que podem oferecer.

RF05 – O sistema deve permitir visualizar os serviços disponíveis.

RF06 – O sistema deve permitir solicitar um serviço.

RF07 – O sistema deve registrar as solicitações realizadas.

## Requisitos Não Funcionais

RNF01 – O sistema deve ser acessível via navegador web.

RNF02 – O sistema deve utilizar Django no backend.

RNF03 – O banco de dados inicial será SQLite.

RNF04 – O sistema deve possuir autenticação de usuários.

RNF05 – O código deve ser organizado e documentado.
