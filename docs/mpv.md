# MVP do Conecta Aluno

## Ideia principal
O Conecta Aluno é uma plataforma web acadêmica para troca de serviços entre estudantes universitários.

## Objetivo do MVP
Criar uma versão inicial simples do sistema, com as funções essenciais para demonstrar a proposta.

## Funcionalidades iniciais
- cadastro de usuário
- login
- criação de perfil
- cadastro de serviços
- listagem de serviços
- solicitação de serviços

## Foco desta primeira versão
Validar a ideia do sistema antes de adicionar funções mais avançadas.
