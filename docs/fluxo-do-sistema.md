# Fluxo do Sistema – Conecta Aluno

## Fluxo principal do usuário

1. O usuário acessa o sistema.
2. O usuário realiza cadastro.
3. O usuário faz login.
4. O usuário cria ou completa seu perfil.
5. O usuário cadastra um serviço que pode oferecer.
6. O usuário visualiza os serviços disponíveis.
7. O usuário solicita um serviço.
8. O sistema registra a solicitação.

## Fluxo principal resumido

Cadastro → Login → Perfil → Cadastro de serviço → Listagem de serviços → Solicitação
