# Conecta Aluno

Plataforma web acadêmica para troca de serviços entre estudantes universitários.

## Objetivo

Permitir que estudantes possam oferecer e solicitar ajuda entre si, compartilhando habilidades como:

- programação
- design
- monitoria
- edição de trabalhos
- suporte em disciplinas

## Tecnologias (MVP)

- HTML
- CSS
- JavaScript
- Python
- Django
- SQLite
